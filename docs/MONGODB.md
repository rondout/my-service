# MongoDB 使用记录

## mongoose 方法

| 功能              | 实现                                                                                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 查询              | Model.find()                                                                                                                                                           |
| 查询-排除某些字段 | Model.find().select(["-xxxx", "-xxxxx", ...])                                                                                                                          |
| 查询-关联查询     | Model.find().populate("关联的 id 字段名", [可选，关联的字段]); **注意：** 这里关联的字段 Id 名在 model 里面必须是 ObjectId 类型，并且用 ref 字段指向所引用的数据库表名 |
